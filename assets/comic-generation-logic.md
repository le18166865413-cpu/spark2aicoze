## 📝 文字处理逻辑

### 1️⃣ 故事拆分 (`splitStoryToScenes`)

#### 拆分策略（三层降级）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        故事拆分流程                                        │
│                                                                             │
│   用户输入: "从前有座山。\n\n山里有座庙。庙里有个老和尚。"                  │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │ 第一层: 按段落拆分 (\n\n+)                                        │      │
│   │                                                                   │      │
│   │ 输入: "从前有座山。\n\n山里有座庙。庙里有个老和尚。"              │      │
│   │ 结果: ["从前有座山。", "山里有座庙。庙里有个老和尚。"]             │      │
│   │                                                                   │      │
│   │ ✅ 如果段落数 >= pageCount，直接使用                               │      │
│   │ ❌ 如果段落数 < pageCount，进入第二层                              │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                    ↓                                        │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │ 第二层: 按句子拆分 ([。！？.!?]+)                                  │      │
│   │                                                                   │      │
│   │ 输入: "从前有座山。山里有座庙。庙里有个老和尚。"                   │      │
│   │ 结果: ["从前有座山", "山里有座庙", "庙里有个老和尚"]              │      │
│   │                                                                   │      │
│   │ 合并句子凑成场景:                                                 │      │
│   │ pageCount=3 → 每页需要1句话                                       │      │
│   │ pageCount=2 → 每页需要2句话                                       │      │
│   │                                                                   │      │
│   │ 公式: sentencesPerScene = ceil(总句子数 / pageCount)              │      │
│   │                                                                   │      │
│   │ 结果: ["从前有座山。", "山里有座庙。", "庙里有个老和尚。"]        │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                    ↓                                        │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │ 第三层: 兜底处理（段落数不足时）                                   │      │
│   │                                                                   │      │
│   │ 循环使用已有段落，添加页码标记:                                    │      │
│   │ ["第1页：从前有座山。", "第2页：从前有座山。", "第3页：从前有座山。"] │
│   └─────────────────────────────────────────────────────────────────┘      │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 代码实现

```typescript
function splitStoryToScenes(storyText: string, pageCount: number): string[] {
  
  // 第一层: 按段落拆分
  const paragraphs = storyText.split(/\n\n+/).filter(p => p.trim())
  if (paragraphs.length >= pageCount) {
    return paragraphs.slice(0, pageCount)  // ✅ 直接返回
  }

  // 第二层: 按句子拆分
  const sentences = storyText.split(/[。！？.!?]+/).filter(s => s.trim())
  if (sentences.length >= pageCount) {
    const scenes: string[] = []
    const sentencesPerScene = Math.ceil(sentences.length / pageCount)
    
    // 分组合并
    for (let i = 0; i < pageCount; i++) {
      const start = i * sentencesPerScene
      const end = Math.min(start + sentencesPerScene, sentences.length)
      const sceneSentences = sentences.slice(start, end)
      scenes.push(sceneSentences.join('。') + '。')
    }
    return scenes
  }

  // 第三层: 兜底
  const scenes: string[] = []
  for (let i = 0; i < pageCount; i++) {
    const baseText = paragraphs[i % paragraphs.length] || sentences.join('。')
    scenes.push(`第${i + 1}页：${baseText}`)
  }
  return scenes
}
```

## 🎨 提示词构建逻辑

### 2️⃣ 提示词组合 (`prompt` 构建)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        提示词构建流程                                        │
│                                                                             │
│   用户输入:                                                                 │
│   ├─ storyText: "从前有座山"                                                │
│   └─ style: "日式漫画"                                                     │
│                                                                             │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │ 步骤1: 从拆分结果取场景文本                                        │      │
│   │   scenePrompt = "从前有座山"                                       │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                    ↓                                        │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │ 步骤2: 构建风格提示词                                             │      │
│   │                                                                   │      │
│   │ if (用户指定了风格)                                               │      │
│   │   styleHint = "，画风：日式漫画风格漫画"                          │      │
│   │ else                                                             │      │
│   │   styleHint = "漫画风格，清晰的分镜构图"                          │      │
│   └─────────────────────────────────────────────────────────────────┘      │
│                                    ↓                                        │
│   ┌─────────────────────────────────────────────────────────────────┐      │
│   │ 步骤3: 拼接完整 prompt                                            │      │
│   │                                                                   │      │
│   │ finalPrompt = scenePrompt + styleHint                             │      │
│   │             = "从前有座山，画风：日式漫画风格漫画"                 │      │
│   └─────────────────────────────────────────────────────────────────┘      │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 代码实现

```typescript
for (let i = 0; i < scenes.length; i++) {
  const scenePrompt = scenes[i]  // 从拆分结果获取
  
  // 构建风格提示词
  const styleHint = body.style 
    ? `，画风：${body.style}风格漫画`     // 用户指定风格

    model: 'gpt-image-2',
    prompt: `${scenePrompt}${styleHint}`,  // 拼接

    replyType: 'json',
  })
}
```

```typescript
const response = await callGrsaiApi({
  model: GRSI_API_CONFIG.model.image2,
  prompt: `${scenePrompt}${styleHint}`,
  aspectRatio: ASPECT_RATIOS.comic,
  replyType: GRSI_API_CONFIG.defaultReplyType,

// 状态判断
if (response.status === 'succeeded' && response.results) {
  // ✅ 成功
  generatedImages.push({
    id: i,
    url: response.results[0].url,  // 获取图片URL
  })
  results.push({
    id: i,
    url: response.results[0].url,
    sceneText: scenePrompt,
  })
} else if (response.status === 'violation') {
  // ⚠️ 违规
  results.push({
    id: i,
    url: null,
    error: "内容包含违规信息",
  })
} else {
  // ❌ 失败
  results.push({
    id: i,
    url: null,
    error: response.error || "生成失败",
  })
}
```

## ⚠️ 当前逻辑的局限性

| 环节 | 当前实现 | 局限性 |
|------|---------|--------|
| **故事拆分** | 按段落/句子机械拆分 | 无法理解语义，可能把一句完整的话拆开 |
| **风格统一** | 通过 styleHint 关键词 | AI 生成是独立的，画风可能不一致 |
| **角色一致性** | 无 | 人物外貌、服装可能每页不同 |
| **分镜设计** | 交给 AI 自由发挥 | 无法控制是单格/多格漫画 |

## 🚀 优化建议

如需改进，可以考虑：

### 方案 A：使用参考图（推荐）

```typescript
// 首先生成封面/第一张图
const firstFrame = await callGrsaiApi({
  model: 'gpt-image-2',
  prompt: `${scenes[0]}${styleHint}`,
  aspectRatio: '1024x1024',
  replyType: 'json',
})

// 后续图片以第一张为参考
for (let i = 1; i < scenes.length; i++) {
  await callGrsaiApi({
    model: 'gpt-image-2',
    prompt: `${scenes[i]}${styleHint}`,
    images: [firstFrame.results[0].url],  // 参考图
    aspectRatio: '1024x1024',
    replyType: 'json',
  })
}
```

### 方案 B：锁定角色描述

```typescript
// 从故事中提取主角特征，添加到每页 prompt
const characterDesc = extractCharacterDescription(storyText)
// 例如: "同一个黑发少年，身穿蓝色T恤"

// 每页都包含角色描述
prompt = `角色设定：${characterDesc}。场景：${sceneText}。${styleHint}`
```

### 方案 C：接入 LLM 语义分析

使用 LLM 对故事进行语义分析，生成：
